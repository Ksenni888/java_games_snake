# Игра "Змейка"
Как запустить игру? 
1. Скачайте архив с файлами и запустите через IDE
2. Скачайте jar-файл snake-1.0-SNAPSHOT-jar-with-dependencies.jar.   
   Положите, например, в корень диска С.   
   Вызовите командную строку win+r -> cmd.   
   Далее убедитесь, что находитесь  вкорне диска С.
   Введите команду: C:\> java -jar snake-1.0-SNAPSHOT-jar-with-dependencies.jar

# Описание игры
Управление с помощью стрелок.
При поедании яблока, длина змейки увеличивается. 
При поедании кактуса - уменьшается.
Если змейка врезается в стену или в саму себя, то GAME OVER.
Приятной игры!
