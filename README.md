# Классическая игра "Змейка" на Java с использованием игрового фреймворка FXGL.
Как запустить игру? 
1. Скачайте архив с файлами и запустите через IDE
2. Скачайте jar-файл snake.jar.   
   Положите, например, в корень диска С.   
   Вызовите командную строку win+r -> cmd.   
   Далее убедитесь, что находитесь  вкорне диска С.
   Введите команду: C:\> java -jar snake.jar

# Описание игры
Стрелки - управление змейкой

🍎 Яблоко +1 длина

🌵 Кактус -1 длина

Если змейка врезается в стену или в саму себя, то GAME OVER.
Приятной игры!
